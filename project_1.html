PROJECT 1

Aplikasi Manajemen Inventaris

Ini adalah aplikasi berbasis web yang dibangun menggunakan Flask, Bootstrap, dan Mysql yang terintegrasi dengan database. Proyek Aplikasi Manajemen Inventaris ini sebagai contoh proyek web development sederhana yang dapat mengelola stok barang, menambah, mengedit, dan menghapus barang dari inventaris.

Aplikasi menjadi proyek saat saya freelance tutor web development untuk siswa-siswi SMK 3 Metro jurusan Rekayasa Perangkat Lunak, saat mereka sedang melakukan Praktik Kerja Industri (PRAKERIN) di Universitas Dharma Wacana Metro.

Fitur

  • Manajemen Barang (CRUD)
  • Kategori Barang
  • Responsive Design (Bootstrap)
  • Monitoring stok

Tech

  • Flask (Python Framework)
  • Mysql (Client Server)
  • HTML, JavaScript, dan Bootstrap
  • Git & GitHub
  • Vercel
